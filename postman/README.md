# Postman files

## Full collection (standard Postman)

- `atr-group-read.postman_collection.json`
- Contains all requests in one collection with folders.

## Light mode (one request per file)

- Use `light/*.curl` when your tooling handles smaller request imports better.
- Start with `light/README.md` for quick usage instructions.

## Shared variables used by these files

- `baseUrl`
- `accessToken`
- `groupId`
- `patientId`
- `coverageId`
- `practitionerRoleId`
- `practitionerId`
- `providerOrganizationId`
- `payerOrganizationId`
- `missingId`
